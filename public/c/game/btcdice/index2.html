2、如何玩BTC-DICE骰子游戏？

玩家可以选择获胜机率以及投注金额来确定投注的赔率与获胜后的盈利。这是骰子游戏的操作界面：

本图代表玩家以49.5%的获胜机率，投注1LTC，如果获胜将盈利1LTC。玩家点击蓝色方框的押大后，会随机生成一个0.0001-99.9999之间的幸运数，如果幸运数大于50.4999,则玩家获胜，盈利1LTC，如果幸运数小于50.4999则玩家失败，亏损1LTC。

什么是获胜机率：代表您有多大的机率会获胜。

什么是赔率：赔率是指假如您获胜后，返给您的币数是您投注额的多少倍。

什么是投注金额：您下注的金额就是投注金额。

什么是获胜盈利：获胜盈利和赔率有关，假如您投注1LTC，赔率是2倍，那么假如您获胜后，您的获胜盈利就是1LTC。

6、SHA256算法可以解密吗？

安全散列算法SHA（Secure Hash Algorithm）是美国国家安全局 （NSA） 设计，美国国家标准与技术研究院（NIST） 发布的一系列密码散列函数，包括 SHA-1、SHA-224、SHA-256、SHA-384 和 SHA-512 等变体。主要适用于数字签名标准（DigitalSignature Standard DSS）里面定义的数字签名算法（Digital Signature Algorithm DSA）。SHA-256是应用广泛的算法，已经经历了实践的检验，至少在目前为止，都是安全暂不能被解密的。
